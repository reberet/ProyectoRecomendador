# Sistema de Recomendaciones para Chatbot

Proyecto desarrollado como parte de un ejercicio práctico de clase.

## Descripción

Primer acercamiento a la construcción del núcleo de un sistema de recomendaciones orientado a chatbots, desarrollado en **C#** con **.NET** y **VSCode**.

El objetivo no fue construir el bot completo, sino sentar las bases del sistema modelando sus entidades principales y su lógica central.

## ¿Qué incluye el proyecto?

El sistema cubre los siguientes componentes esenciales:

- **Usuarios** – representación de los perfiles que interactúan con el sistema
- **Ítems** – elementos susceptibles de ser recomendados
- **Preferencias** – asociación entre usuarios e ítems según sus intereses
- **Historial** – registro de las interacciones previas de cada usuario
- **Recomendaciones** – lógica para sugerir ítems relevantes en base a los datos anteriores


## Contexto

Este proyecto fue realizado como primer acercamiento práctico a los sistemas de recomendación dentro del ámbito académico. El foco estuvo puesto en el diseño y la estructura del núcleo del sistema, más que en la implementación de un chatbot funcional completo.
